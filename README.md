# Neon Dodger

A small neon-styled arcade dodger. Single HTML file, no build step, no dependencies.

**Play:** open `index.html` in any modern browser, or visit the live site.

## Controls

- `←` `→` or `A` `D` — move
- `Space` — dash
- Any key — start / retry

## Goal

Dodge the asteroids, grab the gold orbs (+100). Difficulty ramps with score. Three lives.

## Development

It's one file. Edit `index.html`, refresh the browser. That's the loop.

## Deploy

Pushed to `main` auto-deploys to GitHub Pages.
